# yhiraki/homebrew-tap

Homebrew formulae for my tools.

```sh
brew install yhiraki/tap/dualtap
```

To install the latest `main` instead of the tagged release:

```sh
brew install --HEAD yhiraki/tap/dualtap
```

## Formulae

- **[dualtap](https://github.com/yhiraki/dualtap)** — record your microphone and your Mac's system audio at the same time, natively, without BlackHole or other virtual audio devices.
